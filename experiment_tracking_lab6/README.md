# W&B Experiment Tracking Lab

This lab demonstrates experiment tracking using **Weights & Biases (W&B)** with a Keras neural network.

## Summary of What I Changed

I replaced the original XGBoost-based dermatology classifier with a **Keras deep learning model** trained on the **Breast Cancer Wisconsin** dataset and updated the entire workflow to ensure compatibility with TensorFlow/Keras 3 and Weights & Biases (W&B).

### 1. Replaced XGBoost with a Keras Sequential model
The original notebook used `xgboost.DMatrix`, `xgb.train()`, and `wandb.xgboost.WandbCallback()`. I removed all of that and built a Keras Sequential model trained with `model.fit()`.

### 2. Changed the dataset
Instead of the UCI Dermatology dataset, I used the **Breast Cancer Wisconsin** dataset from sklearn (569 samples, 30 features, 2 classes).

### 3. Updated the W&B callback for Keras 3 compatibility
Keras 3 removed the `save_format` argument, which caused W&B to crash. I disabled all model saving features inside the callback:
```python
wandb.keras.WandbCallback(
    log_model=False,
    save_model=False,
    save_graph=False
)
```

### 4. Replaced XGBoost evaluation with Keras predictions
Updated to use `model.predict()` with softmax + argmax to get class predictions, then computed accuracy/error using NumPy.

### 5. Changed the confusion matrix logging
Switched from `wandb.sklearn.plot_confusion_matrix()` to `wandb.plot.confusion_matrix()` for Keras-compatible logging.

## Before vs After

| Area | Original Code | My Updated Code |
|------|--------------|-----------------|
| Model Type | XGBoost multiclass | Keras Sequential neural network |
| Dataset | Dermatology (UCI) | Breast Cancer Wisconsin (sklearn) |
| Data Pipeline | `xgb.DMatrix` | NumPy arrays with StandardScaler |
| Training | `xgb.train()` | `model.fit()` |
| W&B Callback | `wandb.xgboost.WandbCallback()` | `wandb.keras.WandbCallback()` (saving disabled) |
| Evaluation | `bst.predict()` | `model.predict()` + softmax/argmax |
| Confusion Matrix | `wandb.sklearn.plot_confusion_matrix()` | `wandb.plot.confusion_matrix()` |

## Model Details
- **Dataset:** 569 samples, 30 features, 2 classes (malignant/benign)
- **Model:** Dense(64) → Dropout(0.3) → Dense(32) → Dropout(0.3) → Dense(2, softmax)
- **Optimizer:** Adam (lr=0.001)
- **Epochs:** 20
- **Test Accuracy:** 97.66%
- **Error Rate:** 2.34%
